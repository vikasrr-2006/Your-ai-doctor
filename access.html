How It Works

Browser → Apache (XAMPP, port 80) → PHP Proxy → Flask (port 5000)

The application uses a PHP proxy (proxy.php) to bridge requests between XAMPP's Apache server and the Flask backend.

Setup Instructions

Step 1: Start XAMPP

  1. Open the XAMPP Control Panel
  2. Click Start next to Apache
  3. Ensure Apache status shows green/running

Step 2: Start the Flask Application

  1. Double-click start_flask.bat, OR
  2. Open a terminal and run:
    cd "c:\xampp\htdocs\new help\Your AI Doctor"
    python app.py
  3. Wait for: Running on http://127.0.0.1:5000

Step 3: Access the Application

  1. Open your browser
  2. Navigate to: http://localhost/new%20help/Your%20AI%20Doctor/
  3. If Flask is running, you will be redirected to the app
Tip: You can also access the Flask app directly at http://localhost:5000 without XAMPP.

Troubleshooting

Flask Application Not Running

The Flask backend has not been started. Run start_flask.bat or execute python app.py in the project directory.

Port 5000 Already in Use

Another process is using port 5000. Find and kill it:

netstat -ano | findstr :5000

taskkill /PID <PID> /F

Apache Not Starting (Port 80 Conflict)

Another service is using port 80. Change Apache port in httpd.conf from Listen 80 to Listen 8080, then access via port 8080.

PHP cURL Extension Not Enabled

The proxy requires PHP cURL. In C:\xampp\php\php.ini, uncomment extension=curl and restart Apache.

Missing Python Dependencies

Install required packages:

pip install flask numpy scikit-learn geopy

Requirements

Before running the application, ensure you have:
  • Python 3.8 or higher
  • XAMPP with Apache running
  • Flask, NumPy, scikit-learn installed via pip
